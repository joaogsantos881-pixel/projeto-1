Dúvidas: 
Qual é a diferença entre Git e GitHub? 
O que é um fork? 
O que é um commit e como escrever boas mensagens?

Descobertas:
Git é o sistema de controle de versão; GitHub é o serviço online que hospeda repositórios Git.
É uma cópia de um repositório no seu perfil, que permite modificar o projeto sem alterar o original.
Repositórios públicos são visíveis a todos; privados só a quem você autoriza.
